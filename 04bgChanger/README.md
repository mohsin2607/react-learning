# Background Color Changer

A simple React application that changes the background color with a single button click.

## Features
- Multiple Background Colors
- One-click Color Change
- Responsive UI

## Concepts Learned
- React useState
- onClick Event
- Inline Styling
- Tailwind CSS

## Tech Stack
- React
- Vite
- Tailwind CSS
